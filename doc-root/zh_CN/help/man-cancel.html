cancel(1)

名称

  cancel - 取消任务

概要

  cancel [-E] [-Uusername] [-a] [-hhostname[:port]] [-uusername] [-x] [id] [destination] [destination-id]

说明

  cancel 命令取消打印任务。如果未指定任何 destination 或 id ，则默认目标上的当前打印任务将被取消。

选项

  cancel 可以识别以下选项:

-a
取消指定目标上的所有任务，或取消所有目标上的所有任务（如果未提供）。
-E
连接到服务器时强制加密。
-h hostname[:port]
指定备用服务器。
-U username
指定连接到服务器时要使用的用户名。
-u username
取消 username 所拥有的任务。
-x
除取消外，还删除任务数据文件。

遵循

  与 System V 打印系统不同， CUPS 允许打印机名称包含除空格、制表符、“/” 或 “#” 之外的任何可打印字符。此外，打印机名和类名 不 区分大小写。

例子

  取消当前打印任务： 

cancel

  取消 "myprinter-42" 的任务: 

cancel myprinter-42

  取消所有任务： 

cancel -a

提示

  管理员希望通过 -u 选项防止未经授权的任务取消，应该对 cupsd.conf 中的 Cancel-Jobs 操作进行身份验证。
