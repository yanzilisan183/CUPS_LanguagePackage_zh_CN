lp(1)

名称

  lp - 打印文件

摘要

  lp [-E] [-U username] [-c] [-d destination[/instance]] [-h hostname[:port]] [-m] [-n num-copies] [-o option[=value]] [-q priority] [-s] [-t title] [-H handling] [ -P page-list] [--] [file(s)]
  lp [-E] [-U username] [-c] [-h hostname[:port]] [-i job-id] [-n num-copies] [-o option[=value]] [-q priority] [-t title] [-H handling] [-P page-list]

说明

  lp 提交文件以进行打印或更改挂起的任务。使用 “-” 文件名从标准输入强制打印。

默认目标

  CUPS 提供了许多设置默认目标的方法。首先参考 LPDEST 和 PRINTER 环境变量。如果两者都未设置，则使用当前默认设置（使用 lpoptions(1) 命令），然后使用默认设置（使用 lpadmin(8) 命令）。

选项

  以下选项由 lp 识别：

--
标志着选项的结束；使用此选项打印名称以破折号（ - ）开头的文件。
-E
连接到服务器时强制加密。
-U username
指定连接到服务器时要使用的用户名。
-c
此选项仅用于向后兼容。在支持该选项的系统上，此选项强制打印文件在打印之前复制到假脱机目录。在 CUPS 中，打印文件始终通过具有相同效果的 IPP 发送到调度程序。
-d destination
将文件打印到指定的打印机。
-h hostname[:port]
选择备用服务器。
-i job-id
指定要修改的现有任务。
-m
任务完成时发送电子邮件。
-n copies
设置要打印的份数。
-o "name=value [ ... name=value ]"
设置一个或多个任务选项。请参阅下面的“常见任务选项”。
-q priority
将任务优先级从 1 （最低）设置为 100 （最高）。默认优先级为 50 。
-s
不报告生成的任务 ID （静默模式）。
-t "name"
设置任务名称。
-H hh:mm
-H hold
-H immediate
-H restart
-H resume
指定打印任务的时间。值 immediate 将立即打印文件，值 hold 将无限期保留任务， UTC 时间值（ HH:MM ）将保留任务直到指定的 UTC （非本地）时间。将值 resume 与 -i 选项一起使用，以恢复保留的任务。使用值 restart 和 -i 选项重新启动已完成的任务。
-P page-list
指定要在文档中打印的页面。该列表可以包含由逗号分隔的数字和范围（ #-# ）列表，例如 “1,3-5,16” 。页码指的是输出页面，而不是文档的原始页面， “number-up” 等选项可能会影响页面的编号。

常见任务选择

  除了由 lpoptions(1) 命令报告的 printer-specific 选项外，还提供以下通用选项：

-o job-sheets=name
打印包含文档的封面（ banner ）。“name” 可以是 “classified” 、 “confidential” 、 “secret” 、 “standard” 、 “topsecret” 或 “unclassified” 。
-o media=size
将页面大小设置为 size 。大多数打印机至少支持尺寸名称 “a4” 、 “letter” 和 “legal” 。
-o number-up={2|4|6|9|16}
在每个输出页面上打印 2 、 4 、 6 、 9 或 16 个文档（输入）页面。
-o orientation-requested=4
在横向打印任务（逆时针旋转 90 度）。
-o orientation-requested=5
在横向打印任务（顺时针旋转 90 度）。
-o orientation-requested=6
以反向纵向打印任务（旋转 180 度）。
-o print-quality=3
-o print-quality=4
-o print-quality=5
指定输出质量 － 草稿(3) 、正常(4) 或 最佳(5) 。
-o sides=one-sided
打印在纸的一面。
-o sides=two-sided-long-edge
打印在纸张的两面，用于纵向输出。
-o sides=two-sided-short-edge
打印在纸张的两面，用于横向输出。

遵循

  与 System V 打印系统不同， CUPS 允许打印机名称包含除空格、制表符、 “/” 或 “#” 之外的任何可打印字符。此外，打印机名和类名不区分大小写。

  -q 选项接受的值范围与 Solaris lp 命令不同，与 IPP 任务优先级值（ 1-100 ， 100 为最高优先级）匹配，而不是与 Solaris 值（ 0-39 ， 0 为最高优先级）匹配。

例子

  将文档的两份副本打印到默认打印机： 

lp -n 2 filename

  将双面 legal 文件打印到名为 “foo” 的打印机上： 

lp -d foo -o media=legal -o sides=two-sided-long-edge filename

  将演示文稿文档打印到名为 “bar” 的打印机上： 

lp -d bar -o number-up=2 filename
