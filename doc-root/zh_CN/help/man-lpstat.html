lpstat(1)

名称

  lpstat - 打印 cups 状态信息

摘要

  lpstat [-E] [-H] [-U username] [-h hostname[:port]] [-l] [-W which-jobs] [-a [destination(s)]] [-c [class(es)]] [-d] [-e] [-o [destination(s)]] [-p [printer(s)]] [-r] [-R] [-s] [-t] [-u [user(s)]] [-v [printer(s)]]

说明

  lpstat 显示有关当前类、任务和打印机的状态信息。在不带参数运行时， lpstat 将列出当前用户排队的活动任务。

选项

  lpstat 命令支持以下选项：

-E
连接到服务器时强制加密。
-H
显示服务器主机名和端口。
-R
显示打印任务的排名。
-U username
指定备用用户名。
-W which-jobs
指定要显示的任务为 “completed” 或 “not-completed”（默认值）。此选项必须出现在 -o 选项和/或任何打印机名称之前，否则将在向调度程序发出的请求中使用默认（ “not-completed” ）值。
-a [printer(s)]
显示打印机队列的接受状态。如果未指定打印机，则列出所有打印机。
-c [class(es)]
显示打印机类和属于它们的打印机。如果未指定任何类，则列出所有类。
-d
显示当前默认目标。
-e
显示本地网络上的所有可用目标。
-h server[:port]
指定备用服务器。
-l
显示打印机、类或任务的长列表。
-o [destination(s)]
显示在指定目标上排队的任务。如果未指定目标，则显示所有任务。
-p [printer(s)]
显示打印机及其是否已启用打印。如果未指定打印机，则列出所有打印机。
-r
显示 CUPS 服务器是否正在运行。
-s
显示状态摘要，包括默认目标、类及其成员打印机的列表，以及打印机及其关联设备的列表。这相当于使用 -d 、 -c 和 -v 选项。
-t
显示所有状态信息。这相当于使用 -r 、 -d 、 -c 、 -v 、 -a 、 -p 和 -o 选项。
-u [user(s)]
显示由指定用户排队的打印任务列表。如果未指定用户，则列出当前用户排队的任务。
-v [printer(s)]
显示打印机及其连接到的设备。如果未指定打印机，则列出所有打印机。

遵循

  与 System V 打印系统不同， CUPS 允许打印机名称包含除空格、制表符、“/” 和 “#” 之外的任何可打印字符。此外，打印机名和类名不区分大小写。

  -h 、 -e 、 -e 、 -U 和 -W 选项是 CUPS 独有的。

  Solaris 的 -f 、 -P 和 -S 选项将被自动忽略。
