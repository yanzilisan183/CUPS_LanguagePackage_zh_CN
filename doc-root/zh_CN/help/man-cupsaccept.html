cupsaccept(8)

名称

  cupsaccept/cupsreject - 接受/拒绝发送到目标的任务

摘要

  cupsaccept [-E] [-Uusername] [ -hhostname[:port]] destination(s)
  cupsreject [-E] [-Uusername] [ -hhostname[:port]] [ -rreason] destination(s)

说明

  cupsaccept 命令指示打印系统接受到指定目标的打印任务。

  cupsreject 命令指示打印系统拒绝到指定目标的打印任务。 -r 选项设置拒绝打印任务的原因。如果未指定，则原因默认为 “Reason Unknown （原因未知）”。

选项

  cupsaccept 和 cupsreject 支持以下选项 ：

-E
连接到服务器时强制加密。
-U username
设置连接到服务器时发送的用户名。
-h hostname[:port]
选择备用服务器。
-r "reason"
设置为拒绝任务的打印机显示的原因字符串。

遵循

  cupsaccept 和 cupsreject 命令分别对应于 System V 打印系统命令 “accept” 和 “reject” 。与 System V 打印系统不同， CUPS 允许打印机名称包含除空格、制表符、 “/” 或 “#” 之外的任何可打印字符。此外，打印机名和类名不区分大小写。

  最后， CUPS 版本可能会根据打印系统配置要求用户输入访问密码。
