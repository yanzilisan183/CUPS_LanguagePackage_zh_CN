CUPS Banner 文件格式

介绍

  本规范描述了用于生成打印任务封面页和 CUPS 测试页的 CUPS 标题文件格式（ application/vnd.CUPS banner ）。格式本身由标题和 UTF-8 文本行组成，其中包含注释或关键字和值： 

#CUPS-BANNER

# What to show on the cover page
Show job-id job-name job-originating-user-name time-at-creation

# The header and footer text
Header Cover Page
Footer Cover Page

# Arbitrary "notice" text
Notice All work and no play makes Johnny a dull boy.
Notice All work and no play makes Johnny a dull boy.
Notice All work and no play makes Johnny a dull boy.
Notice All work and no play makes Johnny a dull boy.

# Images to place below the rest
Image /usr/share/doc/cups/images/cups-icon.png
Image /usr/share/doc/cups/images/smiley.jpg

标准关键字

Footer

页脚处的“页脚文本”

  Footer 键定义位于页面底部的文本。只能指定一个 Footer 键。

Header

页眉处的“页眉文本”

  Header 键定义位于页面顶部的文本。只能指定一个 Header 键。

Image

Image /path/to/image/filename
Image relative/path/in/DocumentRoot/filename

  Image 键定义页脚文本上方居中的图像。多个图像作为一组从左到右居中。图像根据需要进行缩放，以适合标称尺寸为 1"/25cm 的页面。

Notice

Notice 任务信息下方显示的文本。
Notice 要在任务信息下方显示的更多文本。

  Notice 键定义任务信息下方居中的文本行。

Show

Show value value ... value

  Show 键列出显示的任务信息。支持以下值：

  • imageable-area: 当前页面大小的可成像区域
  • job-billing: 任务的帐单信息
  • job-id: 任务 ID
  • job-name: 任务名称
  • job-originating-host-name: 打印任务的计算机
  • job-originating-user-name: 打印任务的用户
  • job-uuid: 任务的 UUID
  • options: 任务提供的选项
  • paper-name: 使用的纸张大小的名称
  • paper-size: 使用的纸张尺寸
  • printer-driver-name: 打印机驱动程序使用
  • printer-driver-version: 驱动程序版本
  • printer-info: 打印机信息
  • printer-location: 打印机位置
  • printer-make-and-model: 打印机驱动程序报告的品牌和型号字符串
  • printer-name: 使用的打印机
  • time-at-creation: 任务提交时
  • time-at-processing: 当前日期和时间
