mime.convs(5)

名称

  mime.convs - cups 的 mime 类型转换文件（已弃用）

说明

  mime.convs 文件定义了可用于将文件从一种格式转换为另一种格式的筛选器。

  标准筛选器支持文本、 PDF 、 PostScript 和多种类型的图像文件。

  附加筛选器在 CUPS 配置目录中扩展名为 .convs 的文件中指定。

  mime.convs 文件中的每一行都是注释行、空白行或筛选行。注释行以 # 字符开头。筛选器行指定源和目标 MIME 类型以及与筛选器和要运行的筛选器相关联的相对成本： 

source/type destination/type cost filter

  source/type 字段指定筛选器使用的源 MIME 媒体类型。

  destination/type 字段指定筛选器生成的目标 MIME 媒体类型。

  cost 字段指定运行筛选器的相对成本。值 100 表示筛选器使用大量资源，而值 0 表示筛选器使用很少的资源。

  filter 字段指定筛选器程序文件名。文件名相对于 CUPS 筛选器目录。

文件

  /etc/cups - 典型的 CUPS 配置目录。

  /usr/lib/cups/filter - 典型的 CUPS 筛选器目录。

  /usr/libexec/cups/filter - macOS 上的 CUPS 筛选器目录。

例子

  定义将 PostScript 文档转换为 CUPS 光栅格式的筛选器： 

application/vnd.cups-postscript application/vnd.cups-raster 50 pstoraster

提示

  CUPS 筛选器已弃用，在未来的 CUPS 功能版本中将不再受支持。不支持 IPP 的打印机可以使用应用程序（如 ippeveprinter(1) ）来支持。
