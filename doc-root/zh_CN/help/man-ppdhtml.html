ppdhtml(1)

名称

  ppdhtml - cups html 摘要生成器（已弃用）

摘要

  ppdhtml [-D name[=value]] [-I include-directory] source-file

说明

  ppdhtml 读取驱动程序信息文件并生成 HTML 摘要页面，该页面列出文件中的所有驱动程序以及支持的选项。此程序已弃用，将在 CUPS 的未来版本中删除。

选项

  ppdhtml 支持以下选项：

-D name[=value]
设置要在源文件中使用的命名变量。它相当于在源文件中使用 #define 指令。
-I include-directory
指定备用包含目录。可以提供多个 -I 选项来添加其他目录。

提示

  PPD 文件已弃用，CUPS 的未来功能版本将不再支持 PPD 文件。不支持 IPP 的打印机可以使用应用程序（如 ippeveprinter(1) ）来支持。
