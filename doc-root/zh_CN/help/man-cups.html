cups(1)

名称

  cups - 基于标准的开源打印系统

说明

  CUPS 是用于从文字处理器、电子邮件阅读器、照片编辑器和 Web 浏览器等应用程序打印的软件。它将应用程序生成的页面描述（在此处放置段落、在此处划线等）转换为打印机可以理解的内容，然后将信息发送到打印机进行打印。

  现在，由于每个打印机制造商的做法都不同，所以打印可能非常复杂。CUPS 尽最大努力向您和您的应用程序隐藏这一点，以便您能够专注于打印，而不是如何打印。一般来说，您只需要在第一次使用打印机时了解打印机的情况，即使是这样，CUPS 也可以自己解决问题。

它是如何工作的？

  第一次打印到打印机时，CUPS 会创建一个队列，以跟踪打印机的当前状态（一切正常、无纸等）以及您打印的任何页面。大多数情况下，队列指向通过 USB 端口直接连接到计算机的打印机，但也可以指向网络上的打印机、Internet 上的打印机或多台打印机，具体取决于配置。无论队列指向何处，对于您和您的应用程序来说，它都与任何其他打印机一样。

  每次打印内容时，CUPS 都会创建一个任务，其中包含要将打印发送到的队列、要打印的文档的名称以及页面描述。任务已编号（队列 1 、队列 2 等），因此您可以在打印任务时监视任务，或者在看到错误时取消任务。当 CUPS 获得打印任务时，它会确定将页面转换为可打印格式的最佳程序（筛选器、打印机驱动程序、端口监视器和后端），然后运行这些程序以实际打印任务。

  打印任务完全打印后，CUPS 将任务从队列中删除，并转到您提交的任何其他任务。任务完成时，或打印过程中出现任何错误时，还可以通过几种不同的方式通知您。

我从哪里开始？

  最简单的启动方法是使用 Web 界面配置打印机。转到 “http://localhost:631” 并选择页面顶部的“管理”选项卡。单击“添加打印机”按钮并按照提示操作。

  当要求您输入用户名和密码时，请输入您的登录用户名和密码或 “root” 用户名和密码。

  添加打印机后，系统将要求您设置打印机的默认打印机选项（纸张大小、输出模式等）。根据需要进行任何更改，然后单击“设置默认选项”按钮保存这些更改。某些打印机还支持自动配置 － 单击“查询打印机默认选项”按钮可自动更新选项。

  添加打印机后，您可以从任何应用程序向其打印。您还可以从维护菜单中选择打印测试页，以打印简单的测试页，并验证一切正常任务。

  您还可以使用 lpadmin(8) 和 lpinfo(8) 命令将打印机添加到 CUPS 。此外，操作系统可能包括图形用户界面，或者在将打印机连接到计算机时自动创建打印机队列。

我如何得到帮助？

  CUPS Web 站点（http://www.CUPS.org）提供对 cups 和 cups-devel 邮件列表、其他文档和资源以及错误报告数据库的访问。大多数供应商还提供在线论坛，为您选择的操作系统询问打印问题。

环境变量

  CUPS 命令使用以下环境变量替代文件的默认位置等。出于安全原因，setuid 程序将忽略以下环境变量：

CUPS_ANYROOT
是否允许任何 X.509 证书根（ Y 或 N ）。
CUPS_CACHEDIR
可以在其中找到半持久缓存文件的目录。
CUPS_DATADIR
可以找到数据文件的目录。
CUPS_ENCRYPTION
默认加密级别（ Always 、 IfRequested 、 Never 、 Required ）。
CUPS_EXPIREDCERTS
是否允许过期的 X.509 证书（ Y 或 N ）。
CUPS_GSSSERVICENAME
用于身份验证的 Kerberos 服务名称。
CUPS_SERVER
CUPS 调度程序的主机名 /IP 地址和端口号（主机名:端口 或 IP地址:端口）。
CUPS_SERVERBIN
可以在其中找到服务器帮助程序、筛选器、后端等的目录。
CUPS_SERVERROOT
服务器的根目录。
CUPS_STATEDIR
可以在其中找到状态文件的目录。
CUPS_USER
指定打印请求的用户名称。
HOME
指定当前用户的主目录。
IPP_PORT
指定 IPP 请求的默认端口号。
LOCALEDIR
指定本地化文件的位置。
LPDEST
指定默认打印队列（ System V 标准）。
PRINTER
指定默认打印队列（ Berkeley 标准）。
TMPDIR
指定临时文件的位置。

遵循

  CUPS 符合 Internet 打印协议版本 2.1，并实现 Berkeley 和 System V UNIX 打印命令。

提示

  CUPS 打印机驱动程序、后端和 PPD 文件已弃用，在未来的 CUPS 功能版本中将不再受支持。不支持 IPP 的打印机可以使用应用程序（如 ippeveprinter(1) ）来支持。
