lpoptions(1)

名称

  lpoptions - 显示或设置打印机选项和默认值

摘要

  lpoptions [-E] [-h server[:port]] -d destination[/instance] [-l]
  lpoptions [-E] [-h server[:port]] [-p destination[/instance]] -o option[=value] ...
  lpoptions [-E] [-h server[:port]] [-p destination[/instance]] -r option
  lpoptions [-E] [-h server[:port]] -x destination[/instance]

说明

  lpoptions 显示或设置打印机选项和默认值。如果未使用 -p 选项指定打印机，则使用默认打印机，如 lp(1) 中所述。

  如果未指定 -l 、 -o 或 -r 选项，则在标准输出上报告当前选项。

  提交任务时，lpoptions 命令设置的选项由 lp(1) 和 lpr(1) 命令使用。

  由 root 用户运行时，lpoptions 获取并设置 /etc/cups/lpoptions 文件中所有用户的默认选项和实例。否则，将在 ~/.cups/lpoptions 文件中管理每个用户的默认值。

选项

  lpoptions 支持以下选项：

-E
与 CUPS 服务器通信时启用加密。
-d destination[/instance]
将用户默认打印机设置为 destination 。如果提供了 instance ，则使用该特定实例。此选项覆盖当前用户的系统默认打印机。
-h server[:port]
使用备用服务器。
-l
列出打印机特定选项及其当前设置。
-o option[=value]
为指定的目标指定一个新选项。
-p destination[/instance]
为后面的任何选项设置目标和实例（如果指定）。如果命名实例不存在，则创建该实例。只能使用 lpadmin(8) 程序创建目标。
-r option
从指定的目标中删除指定的选项。
-x destination[/instance]
删除指定目标和实例的选项（如果指定）。如果命名实例不存在，则此操作不起任何作用。只能使用 lpadmin(8) 命令删除目标。

文件

  ~/.cups/lpoptions - user defaults and instances created by non-root users.

  /etc/cups/lpoptions - system-wide defaults and instances created by the root user.

遵循

  lpoptions 命令是 CUPS 独有的。
