cups-config(1)

名称

  cups-config - 获取 cups api 、编译器、目录和链接信息。

摘要

  cups-config --api-version
  cups-config --build
  cups-config --cflags
  cups-config --datadir
  cups-config --help
  cups-config --ldflags
  cups-config [--image] [--static] --libs
  cups-config --serverbin
  cups-config --serverroot
  cups-config --version

说明

  cups-config 命令允许应用程序开发人员确定编译器和链接器的必要命令行选项，以及筛选器、配置文件和驱动程序的安装目录。所有值都报告到标准输出。

选项

  cups-config 命令接受以下命令行选项：

--api-version
报告当前API版本（主要版本 次要版本）。
--build
报告系统特定的内部版本号。
--cflags
报告必要的编译器选项。
--datadir
报告默认 CUPS 数据目录。
--help
报告程序使用情况消息。
--ldflags
报告必要的链接器选项。
--libs
报告要链接到的必要库。
--serverbin
报告存储筛选器和后端的默认 CUPS 二进制目录。
--serverroot
报告默认 CUPS 配置文件目录。
--static
与 --libs 一起使用时，报告静态库，而不是默认（共享）库。
--version
报告 CUPS 安装的完整版本号（主版本.次版本.修订）。

例子

  显示 CUPS 的当前安装版本： 

cups-config --version

  编译一个简单的单文件 CUPS 筛选器：

    cc `cups-config --cflags --ldflags` -o filter filter.c \
        `cups-config --libs`

不建议使用的选项

  以下选项已弃用，但仍适用于向后兼容：

--image
以前用于将 CUPS 图像库添加到库列表中。
