lpr(1)

名称

  lpr - 打印文件

摘要

  lpr [-E] [-H server[:port]] [-U username] [-P destination[/instance]] [-# num-copies [-h] [-l] [-m] [-o option[=value]] [-p] [-q] [-r] [-C title] [-J title] [-T title] [file(s)]

说明

  lpr 提交文件进行打印。命令行上指定的文件将发送到指定的打印机或默认目标（如果未指定目标）。如果命令行上没有列出任何文件， lpr 从标准输入读取打印文件。

默认目标

  CUPS 提供了许多设置默认目标的方法。首先参考 LPDEST 和 PRINTER 环境变量。如果两者都未设置，则使用当前默认设置（使用 lpoptions(1) 命令），然后使用默认设置（使用 lpadmin(8) 命令）。

选项

  以下选项由 lpr 识别：

-E
连接到服务器时强制加密。
-H server[:port]
指定备用服务器。
-C "name"
-J "name"
-T "name"
设置任务名称/标题。
-P destination[/instance]
将文件打印到指定的打印机。
-U username
指定备用用户名。
-# copies
设置要打印的份数。
-h
禁用 banner 打印。此选项相当于 -o job-sheets=none 。
-l
指定打印文件已针对目标格式化，并且应在不进行筛选的情况下发送。此选项相当于 -o raw 。
-m
在任务完成时发送电子邮件。
-o option[=value]
设置任务选项。请参阅下面的“常见任务选项”。
-p
指定打印文件的格式应为带有日期、时间、任务名称和页码的阴影标题。此选项相当于 -o prettyprint ，仅在打印文本文件时有用。
-q
保留打印任务。
-r
指定在提交命名打印文件后应将其删除。

常见任务选择

  除了由 lpoptions(1) 命令报告的打印机特定选项外，还提供以下通用选项：

-o job-sheets=name
打印包含文档的封面（ banner ）。“name” 可以是 “classified” 、 “confidential” 、 “secret” 、 “standard” 、 “topsecret” 或 “unclassified” 。
-o media=size
将页面大小设置为 size 。大多数打印机至少支持尺寸名称 “a4” 、 “letter” 和 “legal” 。
-o number-up={2|4|6|9|16}
在每个输出页面上打印 2、4、6、9 或 16 个文档（输入）页面。
-o orientation-requested=4
在横向打印任务（逆时针旋转 90 度）。
-o orientation-requested=5
在横向打印任务（顺时针旋转 90 度）。
-o orientation-requested=6
以反向纵向打印任务（旋转 180 度）。
-o print-quality=3
-o print-quality=4
-o print-quality=5
指定输出质量 － 草稿(3) 、正常(4) 或 最佳(5) 。
-o sides=one-sided
打印在纸的一面。
-o sides=two-sided-long-edge
打印在纸张的两面，用于纵向输出。
-o sides=two-sided-short-edge
打印在纸张的两面，用于横向输出。

提示

  CUPS 不支持 -c 、 -d 、 -f 、 -g 、 -i 、 -n 、 -t 、 -v 和 -w 选项，如果使用这些选项，会产生警告消息。

例子

  将文档的两份副本打印到默认打印机： 

lpr -# 2 filename

  将双面 legal 文件打印到名为 “foo” 的打印机上： 

lpr -P foo -o media=legal -o sides=two-sided-long-edge filename

  将演示文稿文档打印到名为 “foo” 的打印机上： 

lpr -P foo -o number-up=2 filename
