cupsd(8)

名称

  cupsd - cups 调度程序

摘要

  cupsd [-ccupsd.conf] [ -f] [-F] [-h] [-l] [-scups-files.conf] [ -t]

说明

  cupsd 是 CUPS 的调度程序。它实现了一个基于 Internet 打印协议 2.1 版的打印系统，并支持各地 IPP 的大部分要求。如果命令行上未指定任何选项，则将使用默认配置文件 /etc/cups/cupsd.conf 。

选项

-c cupsd.conf
使用命名的 cupsd.conf 配置文件。
-f
在前台运行 cupsd ；默认设置是作为 “守护进程” 在后台运行。
-F
在前台运行 cupsd ，但将进程与控制终端和当前目录分离。这对于从 init(8) 运行 cupsd 非常有用（8）。
-h
显示程序使用情况。
-l
当从 launchd(8) 或 systemd(8) 运行此选项时，会将其传递到 cupsd 。
-s cups-files.conf
使用命名的 cups-files.conf 配置文件。
-t
测试配置文件的语法错误。

文件

/etc/cups/classes.conf
/etc/cups/cups-files.conf
/etc/cups/cupsd.conf
/usr/share/cups/mime/mime.convs
/usr/share/cups/mime/mime.types
/etc/cups/printers.conf
/etc/cups/subscriptions.conf

遵循

  cupsd 实现 IPP/2.1 所有必需的属性和操作。它还实现了几个 CUPS 特定的管理操作。

例子

  使用默认配置文件在后台运行 cupsd ： 

cupsd

  测试名为 test.conf 的配置文件： 

cupsd -t -c test.conf

  使用名为 test.conf 的测试配置文件在前台运行 cupsd ： 

cupsd -f -c test.conf
