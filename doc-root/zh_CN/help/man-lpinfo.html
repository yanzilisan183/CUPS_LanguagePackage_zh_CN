lpinfo(8)

名称

  lpinfo - 显示可用设备或驱动程序（已弃用）

摘要

  lpinfo [-E] [-h server[:port]] [-l] [--device-id device-id-string] [--exclude-schemes scheme-list] [--include-schemes scheme-list] [--language locale] [--make-and-model name] [--product name] -m
  lpinfo [-E] [-h server[:port]] [-l] [--exclude-schemes scheme-list] [--include-schemes scheme-list] [--timeout seconds] -v

说明

  lpinfo 列出 CUPS 服务器已知的可用设备或驱动程序。第一个表单（ -m ）列出了可用的驱动程序，而第二个表单（ -v ）列出了可用的设备。

选项

  lpinfo 接受以下选项：

-E
连接到服务器时强制加密。
-h server[:port]
选择备用服务器。
-l
显示设备或驱动程序的“长”列表。
--device-id device-id-string
指定在使用 -m 选项列出驱动程序时要匹配的 IEEE-1284 设备 ID 。
--exclude-schemes scheme-list
指定应从结果中排除的设备或 PPD 方案的逗号分隔列表。静态 PPD 文件使用 “file” 方案。
--include-schemes scheme-list
指定应包含在结果中的设备或 PPD 方案的逗号分隔列表。静态 PPD 文件使用 “file” 方案。
--language locale
指定使用 -m 选项列出驱动程序时要匹配的语言。
--make-and-model name
指定在使用 -m 选项列出驱动程序时要匹配的品牌和型号。
--product name
指定在使用 -m 选项列出驱动程序时要匹配的产品。
--timeout seconds
指定使用 -v 选项列出设备时的超时秒数。

遵循

  lpinfo 命令是 CUPS 独有的。

例子

  列出所有设备： 

lpinfo -v

  列出所有驱动程序： 

lpinfo -m

  列出与 “HP LaserJet” 匹配的驱动程序： 

lpinfo --make-and-model "HP LaserJet" -m

提示

  CUPS 打印机驱动程序和后端已弃用，在未来的 CUPS 功能版本中将不再受支持。不支持 IPP 的打印机可以使用应用程序（如 ippeveprinter(1) ）来支持。
